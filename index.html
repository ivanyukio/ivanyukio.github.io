State of the Art

A mobile app with the objective to connect people from all around the world through street art, with the options to browse street art photos taken by users scrolling through a map and clicking on pins that represent a piece of art and its respective location, pins from places you've been to are different from pins you've never visited before, or using a photo belt to simply see all the pictures without having to navigate a map. Sudoku Solver

This was a group project, we did this with the goal to feed it any unsolved sudoku board and it would return the solved board. Using only logical guesses at first it would solve boards that didn't require any kind of number guessing fairly easy, but then get stuck on boards that required guessing.

Implementing guessing was a lot of fun once we understood the proper way to implement backtracking so that the algorithm would go back to the last valid board once it found an invalid path. Eventually the algorithm even became capable of solving a completely empty sudoku board!
